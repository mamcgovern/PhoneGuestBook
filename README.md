# Phone Guest Book
## Description
A modern audio guest book app designed for weddings and special events. When paired with a vintage-style phone, guests can pick up the receiver and leave heartfelt voice messages that are saved digitally for you to treasure forever. It’s a timeless way to capture laughter, advice, and well-wishes—preserved just as they were spoken.
